3. Pass that EXACT string to transcribe_audio as the audio_path argument.

6. After both tools succeed, confirm the transcript_path from the transcribe_audio result and stop.
7. Do NOT repeat tool calls that already succeeded.
8. If any tool returns an error, report it clearly with the exact error message — do not fabricate a success.
